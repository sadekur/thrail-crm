Thrail CRM is a plugin you can add for your Wordpress site. You can use for your business.


== Description ==
Thrail CRM is a powerful WordPress plugin designed to help you manage your customer relationships efficiently. Capture leads, manage them with ease, and automate email communication. 

> **Note: This plugin requires PHP 7.2 or higher to be activated.**

= Usage =

* Go to your Dashboard after installation and navigate to "Thrail CRM >>" to see this CRM List.


**Key Features:**
- **Optin Form**: Use a shortcode to add an opt-in form anywhere on your site.
- **Lead Management**: Store leads in the database and manage them through the admin panel.
- **Admin List**: Utilize WP_List_Table for an organized lead management interface.
- **AJAX Operations**: Remove and edit/update leads instantly without page reloads.
- **Email Automation**: Send instant emails and schedule emails to be sent within 1 hour.
- **Email Logs**: Keep track of all sent emails for future reference.
- **CSV Exporter**: Export your leads with custom filters in CSV format.


== Installation ==
1. Upload the `thrail-crm` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the provided shortcode `[thrail-crm]` to add the opt-in form to any post or page.

== Frequently Asked Questions ==
= How do I add the opt-in form to my site? =
Simply use the shortcode `[thrail-crm]` in any post, page, or widget.

= Can I customize the opt-in form? =
Yes, you can customize the form by editing the settings in the plugin's admin panel.

= How are emails sent using Thrail CRM? =
You can send instant emails directly from the plugin, and also schedule emails to be sent within an hour.

= What data is stored in the email logs? =
The email logs include the recipient's email address, subject, and the date and time the email was sent.

= How do I export leads to CSV? =
Go to the 'Export' section in the plugin's admin panel and use the filter options to customize your CSV export.
